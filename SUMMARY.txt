MindGuard-ondevice is a privacy-first, on-device mental health monitoring system that computes a real-time Wellness Index from smartphone behavioral signals (typing rhythm, app usage, screen events, response latency). All data is processed locally using TensorFlow Lite and PyTorch Mobile, with zero raw data transmission. Built with FastAPI, SQLite, and Aerich, it features modular ingestion, feature engineering, and clinician-configurable risk thresholds. While 154 critical and 325 moderate issues remain (e.g., model quantization drift, battery drain, baseline drift), the architecture supports rapid iteration and clinician integration. The system is designed for HIPAA/GDPR alignment and targets Android deployment with minimal overhead.
